# StyleSwap Client: The Luxury Frontend

The client-side of StyleSwap is a high-performance React application built with Vite. It is designed to deliver a smooth, editorial-style experience for high-end fashion rentals.

## 📁 Directory Structure

- `src/components/`: Modular UI components organized by role.
  - `Admin/`: Dashboard, User Management, and Global Inventory controls.
  - `SubAdmin/`: Vendor-specific management hubs and asset listing forms.
  - `User/`: Product catalog, detailed editorial views, and secure checkout.
  - `Auth/`: Luxury login and registration modals.
  - `Shared/`: Reusable high-end components (Loader, Header, Sidebar).
- `src/context/`: Global state management using React Context API.
  - `AuthContext`: Manages user identity, roles, and security tokens.
  - `ProductContext`: Handles global product listings and real-time availability.
  - `CartContext`: Manages the luxury acquisition bag and date ranges.
  - `OrderContext`: Tracks active rentals and logistics.
- `src/utils/`: High-performance utility functions (fast image uploads, formatting).

## 🎨 Design System: " संपादकी " (Editorial)

StyleSwap uses a custom design system characterized by:
- **Dark Luxury Palette**: Deep midnight backgrounds with gold and emerald accents.
- **Glassmorphism**: Heavy use of backdrop filters and subtle border glows.
- **Micro-Animations**: Purposeful transitions and pulsating identities.

## 🛠️ Key Technologies

- **Vite**: Ultra-fast build tool and dev server.
- **Tailwind CSS**: Utility-first styling with a focus on custom luxury tokens.
- **Lucide React**: Modern, consistent icon set.
